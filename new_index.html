Welcome to myItinerary!

myIntinery is an web app that allows you to create a printable itinerary through simple search in any given location

  • Begin by entering a city you would like search in (you can change the location any time using the Search bar on the map)
  • Then go ahead and add in a title


*Please note: If you are trying to enter a geographical location (e.g. chicago, china, etc...) or an ethnicity (e.g. chinese, indian, etc...) as a keyword you must also include another word to identify the type of place that you would like to search. e.g. "chinese food" or "japanese temple"
